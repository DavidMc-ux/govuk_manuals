# GOVUK Manuals plugin
GOVUK Manuals contains the code and templates to recreate 'hmrc internal' and 'govuk' manual guidance pages within the [GOV.UK Prototype kit](https://prototype-kit.service.gov.uk/docs/).

## How to install
1. In the terminal navigate to your prototype folder you want to install the GOVUK Manuals plugin in.
2. Run `npm install [path]` where `[path]`
